# SiteGist

This repository contains everything you need to run the SiteGist app locally.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Configure your environment variables in the `.env` file based on `.env.example`.
3. Run the app:
   `npm run dev`
